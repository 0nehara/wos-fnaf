# wos-fnaf
A full port of Five Nights at Freddy's in the game 'Waste of Space' (Roblox).

There are still some bugs and missing some stuff from office camera movement.

Microcontroller needs a touchscreen and 10 speakers
[SCREEN MUST BE 16:9 ASPECT RATIO]
Recommended screen sizes: (X 36 | Y 0.1 | Z 64), (X 18 | Z 32 )
